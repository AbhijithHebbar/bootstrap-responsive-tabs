Bootstrap Responsive Tabs

This is a simple jQuery plugin that switches the default bootstrap tabs to collapsible components (accordions) on the viewports that you choose to. It was build using default Bootstrap 3 classes and markup.

Note: Resize your browser window to see how it works.

  • Profile
  • Messages
